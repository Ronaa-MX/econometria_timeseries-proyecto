# Pronóstico de una serie de tiempo
Proyecto final de pronóstico de una serie de tiempo para la materia de econometría en el periódo 26/1. 

Se trabaja con un dataset de una red espacial que contiene un grafo completo ponderado y una tabla de datos de demanda por semana de cada nodo del año 2020 al 2021.

Se realizo: 

* Creación de una matriz de adyaciencia con 0 en la diagonal en base al *csv* de la lista de vértices de grafo
* Reporte en pdf y html del pronóstico
* Un pronóstico a la demanda total semanal de todos los nodos por el modelo **ARIMA** a través de la metodología *Box-Jenkins*
